# 记账宝
基于Vue2.x开发的PC端的记账工具的一款webApp，名为记账宝。  
主要使用了Vuex配合LocalStorage来对数据进行储存和获取。 可以对记账，计算器，标签更删改查等功能。
  
# 核心技术栈
- Vue
- Element-ui
- Echarts
- Vuex
- Html5
- ES6

## 项目搭建
```
npm install
```

### 项目运行
```
npm run serve
```
